dof − (do-for) executes commands for each item of a list dof executes commands for each item (file or string) of a list Example, convert all mp3s to ogg dof -e *.mp3 do ffmpeg -i %f %b.ogg read from stdin. execute; dof displays what commands would be run, this option executes them. force non-stop; dof stops on error (exit code != 0), this option forces dof to ignore them. plain files only; directories, devices, etc are ignored. directories only; plain-files, devices, etc are ignored. help screen. version and license information. In the ‘commands’ you can use several program’s variables. Variables are recognized by prefix ’%’. the full string (or filename) the basename (no directory, no extension) the dirname (only the directory without the ’/’ suffix) the extension without ’.’ prefix the character ’%’. string processing expression... not used yet. Items ’.’ and ’..’ are ignored. Nicholas Christopoulos
