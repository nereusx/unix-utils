path++ − cleanup path Check directories of path removes non-exists, duplicated, and add directories to system’s PATH. With few words, cleanups the PATH. When it runs, it prints out the corrected path. Example (tcsh): setenv PATH ‘path++‘ Example (bash): export PATH=$(path++) Add new directories at the beginning; otherwise appends Print csh command Print sh command Help screen Program information (version, license, ...)
